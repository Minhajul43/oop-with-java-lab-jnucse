# ▶️ How To Run

Clone the repository:

```bash
git clone https://github.com/Minhajul43/oop-with-java-lab-jnucse.git
```

Go to project folder:

```bash
cd oop-with-java-lab-jnucse
```

Compile Java file:

```bash
javac FileName.java
```

Run program:

```bash
java FileName
```
